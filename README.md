# RAPIDD Technologies C# Assignment

This C# program fetches time entry data from an API, aggregates employee hours, and generates:
- An HTML table showing employee rankings with conditional highlighting for hours < 100
- A pie chart visualizing time distribution percentages

## Outputs

### HTML Table
The program generates `output.html` with a styled table of employee time entries.

### Pie Chart
![Employee Time Distribution](employee-time-distribution.png)

## How to Run
Execute the self-contained executable `Ranjith_C# Assignment` (macOS) or build from source using .NET 9.0.

## Repository Contents
- `ranjith-h c#.cs`: Main program source code
- `c#.csproj`: Project file
- `output.html`: Generated HTML table
- `employee-time-distribution.png`: Generated pie chart
- `single/`: Self-contained executable directory
- `final/`: Published executable
